# VHS Upscaler v0.1.0

First public version of the project.

## Highlights

- Windows GUI for a full VHS restore and upscale workflow
- PAL and NTSC project presets
- QTGMC-based deinterlacing and light cleanup
- Real-ESRGAN NCNN Vulkan integration with multiple test models
- Single-frame preview rendering for model comparison
- Estimated final output resolution shown in the UI
- Auto-detection for local tools
- Optional cleanup of intermediate files after successful runs
- Stop button and safe shutdown handling for active jobs
- PowerShell and batch launchers for systems where unsigned `.exe` files are blocked

## Notes

- The app is currently aimed at local Windows use.
- Third-party tools are not bundled in git.
- Unsigned `.exe` builds may be blocked by Windows Smart App Control.
